# WiSafe - Fake WiFi Detector

## Description

This is a beginner cybersecurity project built using Python.
It scans nearby WiFi networks and analyzes signal strength to estimate the risk level.

## Features

* Scan nearby WiFi networks
* Detect signal strength
* Show risk levels (Low / Medium / High)
* GUI interface using Tkinter
* Color-based alerts

## Technologies Used

* Python
* PyWiFi
* Tkinter

## How to Run

1. Install required libraries:

```
pip install pywifi
pip install comtypes
```

2. Run the program:

```
python wifi_gui.py
```
